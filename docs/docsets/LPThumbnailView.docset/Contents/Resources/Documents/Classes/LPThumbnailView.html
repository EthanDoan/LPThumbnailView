open class LPThumbnailView: UIView

LPThumbnailView

A thumbnail view for displaying images and give context in an application which takes photos/video thumbnails. Simply add to a view controler as a subview.

Public API
  • addImage(_:duration:)

    Adds an image to the images array and performs an animation.

    Will animate with duration length, if duration = 0 will not animate and will just perform changes instantly. Animation style performed depends on animationStyle property.

    Declaration

    Swift

    public func addImage(_ image: UIImage, duration: TimeInterval = 0.4)

    Parameters

    image

    The image to add.
    duration

    How long to animate for, if this is 0, no animation is performed.
  • addImageWithContext(_:duration:)

    Adds an image to the images array and performs an animation with context.

    This animation differs from addImage since it uses a temporary image view that was created outside of this instance. It then animates that temporary image view into the correct position and frame and performs an animation to make it look like the temporary image view was added to this thumbnail view.

    The image from the temporary image view is then used to become the new main thumbail for this view.

    Important

    tempImageView is removed from it’s superview after this function is called

    Declaration

    Swift

    public func addImageWithContext(_ tempImageView: UIImageView, duration: TimeInterval = 1.0)

    Parameters

    tempImageView

    The temporary image view which will be animated.
    duration

    How long to animate for.
